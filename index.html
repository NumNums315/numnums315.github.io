BigParser APIBeta

BigParser API allows developers to build powerful experiences using the Grids within BigParser, and make life simple.

What does it do?

The BigParser API allows you to login to your BigParser Account and Request for Data from any grid, sheet or column.

Learn More

Why use it?

Build your app in Minutes! The API is simple, easy to use, you write little or no code, as we give you open source shell code for most things like Mobile Apps, Alexa Skills and Chatbots

Learn More

Who is it for?

The API is designed keeping in mind the needs of young and budding engineers, and yes if you are an experienced developer you will love it for the time it will save you.

Learn More

Alexa

We wrote "Data", an Alexa "skills library" that helps you get Mortgage interest rate and trends, app just to illustrate how you can use BigParser API to build an Alexa skill, backed by the BigParser Mortgages.grid. We have made the relevant parts of this code open source so the community can extend Data.
Remember, Data is van "library of skills" that works across Alexa, Apple Siri and Google Home.
