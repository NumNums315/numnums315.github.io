Alexa

We wrote "Data", an Alexa "skills library" that helps you get Mortgage interest rate and trends, app just to illustrate how you can use BigParser API to build an Alexa skill, backed by the BigParser Mortgages.grid. We have made the relevant parts of this code open source so the community can extend Data.
Remember, Data is van "library of skills" that works across Alexa, Apple Siri and Google Home.
